# Design System - Application de Gestion des Stocks

## Style Visuel
- **Design moderne SaaS** avec interface épurée et professionnelle
- **Palette de couleurs** : Bleu nuit (#1e293b), Blanc (#ffffff), Gris clair (#f8fafc), Accent bleu (#3b82f6)
- **Typographie** : Inter pour les titres, system-ui pour le corps de texte
- **Mood** : Professionnel, épuré, orienté productivité

## Éléments Visuels
- **Icônes** : Heroicons pour une interface cohérente
- **Animations** : Subtiles transitions CSS et micro-interactions
- **Layout** : Grid responsive avec sidebar navigation
- **Cards** : Ombres douces et bordures arrondies

## Composants Clés
- Tableau de bord avec graphiques ECharts.js
- Tables de données avec pagination
- Modals pour les formulaires
- Badges pour les statuts
- Buttons avec états de chargement
